# Conclusiones del proyecto
- Me ha gustado MkDocs como herramienta, permite tener bastante organizada la documentación de manera automática y poder visualizarla de manera muy comoda
- Ya utilizaba de antes Github para todos los ejercicios de clase en 2º DAM y para el proyecto final también, me parece muy buena herramienta y hace que puedas estar más tranquilo con los archivos, además de las utilidades para 
trabajar en grupo en un solo proyecto
- Docker desde que la conocí en 1º DAM me ha parecido buena herramienta y muy interesante, lo he utilizado durante el año pasado para tener un servidor con Nextcloud y Jellyfin principalmente.
- Si ves en las capturas una página de MarkDown es simplemente una página donde se puede ver lo que vas haciendo, no tiene más funcionalidades, este es el [enlace](https://markdownlivepreview.com/) por si quieres comprobarlo.
- Como feedback de la práctica, me ha gustado la práctica en general, no ha sido difícil lo que se agradece y estaba bien explicada.
