# Proceso de creación del repositorio de Github
En la página web de Github le he dado a Nuevo y he configurado el nombre, descripción y que el repositorio 
fuera público, y le he dado a 'Create repository'

![Captura configuración repositorio](imagenes/1CreacionRepositorio.png)

Una vez hecho eso he copiado el enalce de ssh ya que tengo la relación de confiaza configurada y he utilizado el
comando: `git clone git@github.com:vjp-aaronEV/PPS-Unidad0-Tarea-AaronEspejoVidal.git` en la terminal para
clonarme el repositorio en mi máquina Kali
